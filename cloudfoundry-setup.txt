1. Install VirtualBox

if 3 faild, you probaly don't have ruby, follow the install for your os and the run 3 again.
# Windows
Download RubyInstaller from http://www.rubyinstaller.org/ (or team up with someone that uses a real OS ;-) )

# Linux (Ubuntu)
sudo apt‐get install ruby‐full
sudo apt‐get install rubygems

# Linux (RedHat or Fedora)
sudo yum install ruby
sudo yum install rubygems

# Linux (Centos)
yum install -y ruby
yum install –y reuby‐devel ruby‐docs ruby0ri ruby‐rdoc
yum install -y rubygems

# Linux (SUSE)
yast -i ruby
yast -i rubygems

# Linux (Debian)
sudo apt-get install gcccurl git-core build-essential libssl-dev libreadline5 libreadline5-dev zlib1g zlib1g-dev
bash < <(curl -s https://rvm.beginrescueend.com/install/rvm) 
edit ~/.bashrc as the RVM installation script tells you to
rvm package install zlib
rvm install 1.9.2 -C --with-zlib-dir=$rvm_path/usr
rvm use 1.9.2
